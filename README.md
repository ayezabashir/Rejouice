### Rejouice Clone

A high-performance landing page recreation focusing on fluid typography, sophisticated motion, and a seamless user experience. This project explores the intersection of minimalist design and complex animation orchestration using a lightweight, no-build setup.

---

### ## Project Overview

This project is a frontend implementation of the **Rejouice** landing page. The goal was to replicate the high-end aesthetic and interactive feel of the original site while keeping the codebase accessible and modular. It features custom cursor interactions, smooth scroll integration, and intricate masking effects.

While the design is inspired by the original site, the implementation logic and animation sequences were built from scratch to focus on clean, performant code.

---

### ### Key Features

* **Custom Motion Orchestration:** Detailed timelines for entrance animations and scroll-based triggers.
* **Dynamic Typography:** Implementation of fluid text scaling and SVG masking.
* **Interactive Elements:** Custom cursor behaviors and hover effects that react to user input.
* **Lightweight Setup:** Built using CDNs for a streamlined development process without heavy dependency management.

---

### ## Tech Stack

* **HTML5 & CSS3:** For structural layout and styling.
* **JavaScript (ES6+):** Handling the core interactivity and logic.
* **GSAP (GreenSock Animation Platform):** Powering the motion design, specifically utilizing `ScrollTrigger` for scroll-driven animations.
* **Locomotive Scroll:** To achieve smooth, cinematic scrolling momentum and parallax effects.

### ## Getting Started

This project is designed for simplicity and does not require a package manager.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/rejouice-clone.git
    ```
2.  **Open the project:**
    Open the folder in **Visual Studio Code**.
3.  **Run the Preview:**
    Click the **"Go Live"** button in the bottom-right corner of the status bar (requires the *Live Server* extension) to view the project in your browser.

---

### ## Development Focus

A significant portion of this project involved managing complex animation timelines and ensuring they stay synced with **Locomotive Scroll**. By utilizing CDNs, the focus remained entirely on the frontend implementation and refactoring the JavaScript to handle shared logic efficiently across different sections of the landing page.
