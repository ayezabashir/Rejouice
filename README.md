## Rejouice Clone

A high-performance landing page recreation focusing on fluid typography, sophisticated animations, and a seamless user experience. This project explores the intersection of minimalist design and complex motion orchestration.

---

### ## Project Overview

This repository contains a frontend implementation of the **Rejouice** landing page. The goal was to replicate the high-end aesthetic and interactive feel of the original site while maintaining clean, modular code. It features custom cursor interactions, smooth scroll integration, and frame-by-frame animation sequences.

### ### Key Features

* **Custom Motion Orchestration:** Detailed timelines for entrance animations and scroll-based triggers.
* **Dynamic Typography:** Implementation of fluid text scaling and masking effects.
* **SVG Animations:** Complex vector paths used for branding and decorative elements.
* **Component Architecture:** Built with a focus on reusability and clean state management.
* **Responsive Design:** Fully optimized for all screen sizes, ensuring the motion design translates well to mobile.

---

### ## Tech Stack

* **React:** For building the UI and managing component logic.
* **GSAP (GreenSock Animation Platform):** Powering the core motion, including `ScrollTrigger` and custom timelines.
* **CSS/Tailwind:** For layout and styling.
* **Vite:** For a fast, modern development environment.

### ## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/rejouice-clone.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```

---

### ## Development Process

The development focused on recreating the visual "weight" of the original site. A significant portion of the work involved refactoring components to handle shared logic efficiently, particularly when managing multiple animation triggers across the DOM. While the design is inspired by Rejouice, the underlying logic and implementation were built from scratch to ensure performance and scalability.
